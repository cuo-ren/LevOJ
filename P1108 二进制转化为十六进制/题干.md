# P1108 二进制转化为十六进制

## 题目描述
将二进制数转化为十六进制数.

## 输入描述
一行，一个只包含 0 和 1 的字符串 b，表示需要转化的二进制数.

## 输出描述
输出二进制数对应的十六进制数.

## 样例输入 #1

```
10010
```

## 样例输出 #1

```
12
```

## 样例输入 #2

```
11011
```

## 样例输出 #2

```
1B
```

## 数据范围

对于 30% 的数据，满足字符串 b 的长度不超过 30 位

对于 100% 的数据，满足字符串 b 的长度不超过 200000 位

时间限制  1 s

空间限制  16 MB

测试数据  10文件 单组输入
