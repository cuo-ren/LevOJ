# P1031 三角形的面积

## 题目描述

给定3条线的长度，求出用这3条线组成的三角形的面积。提示：可以使用海伦公式，海伦公式为  $S = \sqrt{p(p-a)(p-b)(p-c)}\$

本题需要正确使用数据类型，避免数值越界或者精度不足。

## 输入描述

多组输入，每组输入为1行，其中有3个整数a,b,c (0≤a,b,c≤10000) , 分别代表3条线的长度。

## 输出描述

对于每组输入，如果这3条线能组成三角形，则输出三角形的面积，结果小数点后保留2位，否则，输出−1.

## 样例输入

```
3 4 5
1 1 10
```

## 样例输出

```
6.00
-1
```

时间限制：1s

空间限制：16MB

测试数据：1文件 多组输入
