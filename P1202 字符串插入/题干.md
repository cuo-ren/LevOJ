# P1202 字符串插入

## 题目描述

有两个字符串 str 和 substr，str 的字符个数不超过 15，substr 的字符个数为 3。（字符个数不包括字符串结尾处的 "反斜杠 0"。）将 substr 插入到 str 中 ASCII 码最大的那个字符后面，若有多个最大则只考虑第一个。

## 输入描述

输入包括一行，格式为

str substr

## 输出描述

输出插入之后的字符串

## 样例输入 1

```
abcab eee
```

## 样例输出 1

```
abceeeab
```

## 样例输入 2

```
12343 555
```

## 样例输出 2

```
12345553
```

时间限制  1 s

空间限制  16 MB

测试数据  6文件 单组输入
