# P1323 青蛙过河

## 题目描述

聪明的青蛙要从河岸到河的另一边去。河面上有一排很规则的荷叶从河岸的一遍延伸到另一边。青蛙可以在荷叶上依次跳过，但是这家伙不想游泳，所以不能从水中游过去。假设，每次最多只能跳过两片荷叶 (当然，一片也是可以的)。荷叶的数量 n<24。这家伙需要知道有多少中跳法。

## 输入描述

每行一个正整数，表示荷叶的数量。

## 输出描述

每行一个结果，青蛙过河的方法数

## 样例输入

```
10
17
21 
```

## 样例输出

```
89
2584
17711
```

时间限制  1 s

空间限制  16 MB

测试数据  1文件 多组输入
