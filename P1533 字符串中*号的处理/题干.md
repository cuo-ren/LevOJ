# P1531 字符串排序

## 题目描述
输入一个只包含字母和 * 号的字符串。请删除除了字符串前导的 * 号之外的其他 * 号。 例如，若字符串中的内容为 `****A*BC*DEF*G*******` , 删除后，字符串中的内容则应当是 `****ABCDEFG` 。

## 输入描述
输入的字符串中只包含字母和 * 号的字符串，长度不超过 10000

## 输出描述
输出处理后的字符串

## 样例输入

```
****A*BC*DEF*G*******
```

## 样例输出

```
****ABCDEFG
```

时间限制  1 s

空间限制  32 MB

测试数据  1文件 单组输入
