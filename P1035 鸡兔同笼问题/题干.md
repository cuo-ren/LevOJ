# P1035 鸡兔同笼问题

## 题目描述
鸡和兔子在被关在一起。鸡有一个头，两条腿；兔子有一个头，四条腿。现已知笼子中头的个数m, 和腿的个数n(1<=m,n<1000) , 求出笼子中鸡和兔子的个数a,b.

## 输入描述
多组输入，每组数据一行，其中包含2个正整数m,n,分别代表笼中头的个数和腿的个数。

## 输出描述
对于每组测试数据输出仅一行，其中包含两个整数a,b, 分别代表鸡的个数和兔子的个数，如果算不出鸡和兔子的个数则输出 -1 -1

## 样例输入

```
1 4
2 4
10 16
```

## 样例输出

```
0 1
2 0
-1 -1
```

时间限制  1 s

空间限制  16 MB

测试数据  1文件 多组输入
